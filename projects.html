pretty-lib

A simple library application, where you are able to create, find, update and remove books and their authors. The project was initially started, when I was reading about HUE colors, which is why the theme is based on HUE colors. The project is made in a way that makes it possible to change the entire color theme by changing 1 variable.
The project follows the MVC architecture and is made by JavaScript, HTML, CSS and NodeJS. The project is currently live on heroku, just hit the Live Demo button to view the project demonstration.
